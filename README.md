A simple test homepage project.

## Description

This is a basic website for testing purposes that includes a homepage with navigation links.

## Files

- `index.html` - Main HTML file with the homepage structure
- `style.css` - Stylesheet for page styling

- Welcome header with navigation menu
- Navigation links (Home, About, Contact)
- Simple styling with custom color scheme

## Usage

Open `index.html` in a web browser to view the homepage.
